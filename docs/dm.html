5.1.1. Mutating Joins¶

  1. Datasets

Python

import pandas as pd
left = pd.DataFrame({'A': ['A0', 'A1', 'A2', 'A3'],
                    'B': ['B0', 'B1', 'B2', 'B3'],
                    'C': ['C0', 'C1', 'C2', 'C3'],
                    'D': ['D0', 'D1', 'D2', 'D3']},
                    index=[0, 1, 2, 3])

right = pd.DataFrame({'A': ['A0', 'A1', 'A6', 'A7'],
                       'F': ['B4', 'B5', 'B6', 'B7'],
                       'G': ['C4', 'C5', 'C6', 'C7'],
                       'H': ['D4', 'D5', 'D6', 'D7']},
                       index=[4, 5, 6, 7])

print(left)
print(right)

R

left = data.frame(A = c('A0', 'A1', 'A2', 'A3'),

  1. Left join

Python

  • Code:
# left join
left_join = left.merge(right, on='A', how='left')
print(left_join)
  • Result:

# left join
